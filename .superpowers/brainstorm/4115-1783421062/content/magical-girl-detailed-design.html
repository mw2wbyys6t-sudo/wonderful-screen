魔法少女 · 变身启动界面 详细方案

以"守护星座召唤"为核心概念，从加载到入口打造连贯的二次元仪式感。

一、核心概念

用户打开网站 = 启动一次"星云召唤仪式"。8 位 ISML 萌王作为守护星座，围绕中央的魔法阵/玻璃圆盘缓缓觉醒。加载完成后，光芒收敛为登录入口，用户点击"开始旅程"正式穿越到动漫宇宙。

三、加载页 LoadingPhase 设计

加载页结构
玻璃圆盘
正在召唤守护星座… 68%

详细动效

  1. 0~30%：深紫背景缓缓显现，中心玻璃圆盘从虚到实旋转出现
  2. 30~70%：8 位角色头像作为"星辰"沿轨道逐个点亮，每个角色亮起时散发对应主题色光芒
  3. 70~100%：所有星辰连线成守护星座图案，魔法阵完全绘制，中央出现标题光效
  4. 加载完成：光芒爆发，整个画面向中心收缩，过渡到登录页

四、登录/入口页 LandingPhase 设计

入口页结构
NEBULA CHRONICLE
星云编年史
手势次元 · 守护星座已集结
中央法阵
开始旅程

详细动效

  1. 入场：从加载页收缩的光芒中展开，星云背景以呼吸节奏轻微缩放
  2. 中央法阵：玻璃圆盘持续缓慢旋转，每隔几秒有一次光扫过效果
  3. 角色守护环：8 位角色围绕法阵排列，头像轻微上下浮动，像守护星座
  4. 标题：手写感字体，字符逐个淡入，伴随星尘粒子飘落
  5. 按钮：宝石形状，hover 时发光增强，点击后星尘向四周爆发并进入下一页

五、字体与文案

  • 标题：保留"星云编年史"中文主标题，副标题改为"手势次元 · 守护星座已集结"
  • 按钮："开始旅程"改为"开启次元之门"或"启动星云召唤"
  • 加载提示："正在召唤守护星座…""星辰连线中…""次元之门即将开启…"

七、技术实现要点

  • 使用 CSS 动画实现魔法阵旋转、光扫、角色呼吸浮动
  • 使用 Canvas 绘制樱花/星尘/羽毛粒子，增强手绘梦幻感
  • 使用 SVG stroke-dashoffset 实现魔法阵描边绘制动画
  • 使用 CSS clip-path 和 filter 实现加载完成时的光芒爆发与收缩转场
  • 视频背景可选，默认用 minimax-nebula-bg.png 做静态背景以保证性能

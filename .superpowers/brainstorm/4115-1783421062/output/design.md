# 星云编年史 · 手势次元 — 加载页 & 登录页 redesign

## 1. 设计方向

采用「魔法少女 · 变身启动界面」风格：
- 二次元视觉语言明确
- 带有手绘/梦幻光效质感
- 以「守护星座召唤仪式」为核心叙事
- 从加载到入口打造连贯的仪式感

## 2. 色彩系统

|---|---|---|
| 深紫夜 | `#1a0b2e` | 主背景 |

## 3. 加载页 LoadingPhase

### 3.1 结构

- 全屏深紫渐变背景
- 底层：`magic-circle.jpg` 作为魔法阵背景，叠加粉紫色光效，缓慢旋转
- 中层：SVG 手绘感魔法阵描边，使用 stroke-dashoffset 实现绘制动画
- 中上层：中心玻璃圆盘（`title-glass-disc.jpg`），旋转 + 呼吸发光
- 外层：8 位角色头像沿轨道分布，作为「星辰」逐个点亮
- 底部：加载进度 HUD + 召唤提示文案

### 3.2 动效分阶段

| 进度 | 动画 |
|---|---|
| 0~30% | 深紫背景淡入，魔法阵背景旋转出现，中心玻璃圆盘从虚到实旋转入场 |
| 30~70% | 8 位角色头像作为星辰，沿轨道按顺序逐个放大、发光、连点成线 |
| 70~100% | 星座连线完成，SVG 魔法阵描边绘制完毕，中央标题光效闪烁 |
| 100% | 光芒爆发，画面向中心收缩，通过 clip-path + scale 过渡到登录页 |

### 3.3 文案

- 正在召唤守护星座…
- 星辰连线中…
- 次元之门即将开启…
- 进度显示：68%

## 4. 登录/入口页 LandingPhase

### 4.1 结构

- 全屏背景：`minimax-nebula-bg.png` 叠加深紫渐变遮罩，轻微呼吸缩放
- 可选：背景播放 `nebula-trailer-v3-a/b/c.mp4` 并叠加紫粉色滤镜
- 顶部：英文小字 NEBULA CHRONICLE
- 中上部：主标题「星云编年史」，手写感强，逐字淡入
- 中部：中央法阵（`title-glass-disc.jpg`），持续缓慢旋转，光扫效果
- 中下部：8 位角色头像围绕法阵排列成守护环，轻微上下浮动
- 底部：宝石形状按钮「开启次元之门」
- 四角：弱化 HUD 装饰线

### 4.2 动效

| 元素 | 动画 |
|---|---|
| 背景 | 呼吸缩放 1.0 → 1.05，20s 循环 |
| 标题 | 字符逐个淡入，伴随星尘飘落 |
| 中央法阵 | 持续旋转，每 4s 一次光扫 |
| 角色守护环 | 上下浮动，周期错开，hover 时放大并显示名字 |
| 按钮 | 宝石发光，hover 增强，点击后星尘向四周爆发 |
| 入场转场 | 从加载页中心光芒展开，元素依次浮现 |

### 4.3 文案

- 顶部英文：NEBULA CHRONICLE
- 主标题：星云编年史
- 副标题：手势次元 · 守护星座已集结
- 按钮：开启次元之门

## 6. 技术实现要点

- CSS 动画：旋转、呼吸、光扫、浮动
- SVG stroke-dashoffset：魔法阵描边绘制
- Canvas：星尘/樱花粒子系统
- clip-path + transform：转场动画
- 保留 DataEngine.load() 逻辑
- 保留 initAudio() 入口音频初始化
- 视频背景默认关闭，可通过配置开启

## 7. 响应式

- 桌面：完整法阵 + 守护环 + 粒子
- 平板：法阵缩小，守护环角色数量可减至 4 个
- 手机：仅保留中央法阵 + 标题 + 按钮，守护环简化

# 星云编年史 · 二次元史诗四幕剧视觉重构设计

## 设计目标

消除 Loading → Showcase → Landing → Universe 四个阶段间的视觉素材重复问题，通过「叙事分章」让每阶段拥有独特的视觉主题、配色和素材分配，使整个流程成为一段情绪递进的二次元史诗体验。

## 叙事四幕总览

```
第一幕·星之召唤(Loading) → 第二幕·英灵殿(Showcase) → 第三幕·次元之门(Landing) → 终章·星海航行(Universe)
  深邃神秘·仪式感           科技全息·庄严集结           史诗高潮·澎湃开启            辽阔自由·沉浸探索
  暗紫+金色                 深空靛蓝+全息青             暗紫蓝+粉紫金渐变             黑+霓虹青/品红
```

情绪曲线：神秘期待 → 震撼检阅 → 史诗高潮 → 辽阔沉浸

## 第一幕：星之召唤（LoadingPhase）

### 情绪
深邃、神秘、期待感。古老召唤仪式启动，符文点亮，星座连线，但守护者尚未现身。

### 视觉层次（从底到顶）
1. **背景**：极深紫黑渐变 `#0a0515 → #1a0b2e`
2. **符文粒子**：Canvas 绘制六芒星/菱形符文粒子缓慢飘浮（非普通圆点星），金色 `#d4a853` 和青白 `#c8e0ff`
3. **魔法阵层**：
   - 底层：`magic-circle.jpg` 中速旋转（60s/圈），`mix-blend-mode: screen`，暗紫+金色调
   - 上层：SVG 手绘法阵反向旋转（40s/圈），三层圆环+五芒星逐笔绘制动画
4. **能量核心**：法阵中心圆形区域内播放 `nebula-trailer-v3-b.mp4`，经 `hue-rotate(280deg) saturate(0.8) brightness(0.6)` 偏暗紫色调，像能量在阵心汇聚
5. **守护星点**：外围8个光点（空光晕，不是角色头像），随进度依次点亮，点亮后连成星座
6. **进度文本**：法阵中心用古文风格显示「召集中…」→「星辰回应」→「门即将开启」，不用百分比环
7. **爆发转场**：100%时中心白光爆发，魔法阵碎裂式缩放消失

### 素材使用
- `magic-circle.jpg`（独此阶段）
- `nebula-trailer-v3-b.mp4`（经调色，独此阶段）
- Canvas 符文粒子（程序化生成）
- SVG 法阵（程序化绘制）
- ❌ 不使用角色头像、不使用 title-glass-disc.jpg

### 配色
- 背景：`#0a0515 → #1a0b2e`
- 主色：金色 `#d4a853`
- 辅助：暗紫辉光 `#7b2fbe`
- 点缀：青白星点 `#c8e0ff`

## 第二幕：英灵殿（ShowcasePhase）

### 情绪
科技感、全息、庄严。从魔法仪式跃迁到全息数据空间，历代高人气作品以全息卡牌形式环绕检阅。

### 视觉层次
1. **背景**：极深靛蓝 `#02040a`，全屏播放 `nebula-trailer-v3-c.mp4`，经 `hue-rotate(180deg) saturate(1.2) brightness(0.4)` 变为青蓝色数据流，低透明度(0.3)
2. **全息网格**：CSS 绘制淡青色透视网格线（地面消失点效果），营造科技空间纵深感
3. **编年史核心**：中心跳动的青色能量球（CSS radial-gradient + pulse），代表数据库核心
4. **全息卡牌环**：保留 3D 旋转布局，卡牌升级：
   - 边框：青色全息边框 + 故障闪烁（glitch）效果
   - 扫描线：`light-streak.png` 或 CSS 线性渐变实现垂直扫描线
   - 数据噪点：边缘添加数字噪点/像素错位
   - 信息：卡牌下方显示作品名/年份/评分的青色全息文字
5. **光环**：每张卡牌光晕沿用流派色（Action红/Romance粉/Sci-Fi青等）
6. **提示**：底部全息文字「历代高人气作品集结中…」→「等待跳过或点击→」

### 素材使用
- `nebula-trailer-v3-c.mp4`（经调色为青蓝数据流，独此阶段）
- `light-streak.png`（全息扫描线）
- 作品封面 0.jpg~41.jpg（全息卡主体）
- ❌ 不使用魔法阵、角色头像、圆盘

### 配色
- 背景：`#02040a`
- 主色：全息青 `#00f3ff`
- 辅助：流派色光晕
- 文字：白色全息文字

## 第三幕：次元之门（LandingPhase）

### 情绪
史诗高潮、澎湃、庄严。所有元素汇聚于此，巨大传送门矗立，8位守护者分列两侧，即将踏上旅程。

### 视觉层次
1. **远景星空**：`minimax-nebula-bg.png` 暗化 + 模糊(8px) 作为远景
2. **传送门主体**：
   - 外框：CSS/SVG 绘制椭圆拱门轮廓，带哥特式装饰，发粉紫光芒，有呼吸光晕动画
   - 门内：`nebula-trailer-v3-a.mp4` 播放星云涌动
   - 核心：门正中央悬浮 `title-glass-disc.jpg` 水晶球，缓慢旋转+呼吸发光
   - 折射：`glass-caustic.jpg` 以 `mix-blend-mode: screen` 叠加，模拟玻璃折射
3. **守护者排列**：8个角色头像分两列竖排于门两侧（左4右4），不是圆周轨道
   - 每人脚下有发光符文圆阵
   - 独立浮动动画，hover放大+光环
4. **标题区**：门上方「星云编年史」大字（保留现有渐变+辉光），下方 `title-ornament.png` 装饰
5. **按钮**：「开启次元之门」在门前（水晶下方），点击后门炸开warp转场
6. **四角 HUD**：
   - 左上「次元坐标：已锁定」
   - 右上「守护者：8/8 集结完毕」
   - 左下「次元能量：满溢」
   - 右下「等待召唤师指令」

### 素材使用
- `title-glass-disc.jpg`（独此阶段，门核心水晶）
- `nebula-trailer-v3-a.mp4`（独此阶段，门内星云）
- `minimax-nebula-bg.png`（独此阶段，远景星空）
- 8个角色头像（独此阶段，守护者分列两侧）
- `glass-caustic.jpg`（门表面折射）
- `title-ornament.png`（标题装饰）
- ❌ 不使用v3-b/v3-c、不使用魔法阵

### 配色
- 背景：`#0d0618`
- 门边框：玫粉 `#ff69b4`
- 核心水晶：青紫渐变 `#b892ff → #00f3ff`
- 点缀：金色文字

## 终章：星海航行（UniversePhase）

### 情绪
辽阔、自由、沉浸。穿越门后视野完全打开，真正驾驶飞船航行在动漫宇宙中。

### 视觉层次
1. **宇宙主体**：保留 3D 粒子宇宙 / 2D 螺旋降级，背景 `nebula-bg.jpg`
2. **HUD 驾驶舱化**：
   - 按钮边框更细更锐利（1px边框，小圆角，像飞船控制台）
   - 左上角小型状态：信号格/能量条/当前年份坐标
   - 搜索框改为「航线搜索」风格
   - 年份导航改为「时间跳跃」旋钮风格
   - 语音/播报按钮保持但风格统一为控制台按钮
3. **NodePanel 情报面板化**：
   - 边框加入扫描线动画
   - 侧边装饰条改为信号强度条
   - 推荐卡片改为雷达触点风格
4. **移除所有装饰层**：没有魔法阵、没有角色环、没有门——纯宇宙+控制台

### 素材使用
- `nebula-bg.jpg`（宇宙背景）
- 作品封面（NodePanel详情）
- 3D/2D 粒子宇宙（现有引擎）
- ❌ 所有前导阶段装饰素材均不出现

### 配色
- 背景：深空黑 `#050714`
- HUD主色：霓虹青 `#00f3ff`
- 选中/警告：品红 `#ff2a6d`
- 辅助：暗紫 `#b892ff`

## 四幕转场设计

| 转场 | 效果 |
|---|---|
| Loading → Showcase | 法阵中心白光爆发(0.4s) → 白光中浮现青色全息网格线(0.3s) → 网格重组为3D卡牌空间 |
| Showcase → Landing | 所有卡牌向中心汇聚(0.5s) → 汇聚点粉红爆炸(0.3s) → 光芒中传送门轮廓显现+守护者淡入 |
| Landing → Universe | 点击按钮后，门从中心向外炸开（现有warp效果增强：白光+星爆+缩放1.25x）→ 白光散尽星空铺满，HUD延迟0.3s淡入 |

## 需要生成/补充的素材

1. **传送门外框**（SVG/CSS 实现为主，若效果不足可生成一张哥特式魔法拱门 PNG 带透明通道）
2. **全息网格线**（CSS 实现，无需图片）
3. **符文粒子形状**（Canvas 绘制，无需图片）
4. **驾驶舱 HUD 装饰元素**（CSS 实现）

优先使用 CSS/SVG/Canvas 程序化生成所有视觉特效，减少对外部图片的依赖，确保性能和可维护性。

## 实现优先级

1. 第一幕 Loading（召唤仪式）—— 移除角色环和圆盘，重做法阵+符文粒子+核心能量
2. 第二幕 Showcase（英灵殿）—— 添加全息网格+数据流背景+卡牌glitch升级
3. 第三幕 Landing（次元之门）—— 重构布局：竖排守护者+椭圆传送门+核心水晶
4. 第四幕 Universe（驾驶舱 HUD）—— HUD和NodePanel视觉风格统一升级
5. 转场效果优化
6. 构建验证+推送

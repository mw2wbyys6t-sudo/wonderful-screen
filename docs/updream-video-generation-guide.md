# UpDream 视频生成指南 · 星云编年史登录页

> 本指南用于在 UpDream 中生成 2–3 个登录页背景视频。
> 目标：人物贴合原作、画面华丽有二次元高级感、适合作为网页背景循环播放。

---

## 一、项目背景

为一个二次元角色企划「星云编年史 NEBULA CHRONICLE」的登录页生成背景视频。

页面流程：
1. 加载画面（星云 + 转圈）
2. 8 位经典动漫女主角以琉璃玻璃卡片形式环绕星门展示
3. 卡片退场后，背景切换为角色视频轮播
4. 出现 Apple 风格「开始旅程」按钮

视频将用于第 3 步的背景轮播。

---

## 二、通用生成参数

| 参数 | 建议值 |
|------|--------|
| 分辨率 | 1920 × 1080（16:9） |
| 时长 | 6–8 秒 |
| 帧率 | 24–30 fps |
| 风格 | 动漫/Cinematic Anime |

| 循环友好 | 首尾画面尽量接近，便于无缝循环 |

- [ ] 首尾画面接近，便于循环

- [ ] 分辨率 1920×1080

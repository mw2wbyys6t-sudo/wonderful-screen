# 🌌 星云编年史 · 手势次元 —— TRAE AI创造力大赛参赛作品

---

## ✨ 创意名称

**星云编年史 · 手势次元 (Nebula Chronicle)**

---

## 📝 创意介绍

### 想解决什么问题？
传统的时间线展示方式枯燥乏味，用户与内容的交互仅限于鼠标点击。我希望打破这种单一的交互模式，让探索历史变成一种**身临其境的宇宙漫游体验**。

### 为什么会想到做这个？
受到Google Gemini手势交互Demo的震撼启发，我思考：为什么不能用同样的方式探索二次元文化？动漫60年的发展史就像一片浩瀚星云，每一部作品都是一颗独特的星辰，值得用最酷的方式去触碰。

### 大概是什么产品？
一个基于**Web浏览器的沉浸式手势交互可视化系统**——用户通过摄像头捕捉手势，在动态星云中"穿越"二次元动漫历史，手指一滑即可探索、点击、查看详情。

## 🛠 技术实现

| 技术 | 用途 |
|------|------|
| **MediaPipe Hands** | 实时手势识别（指向/滑动/捏合/张开） |
| **Three.js + Canvas** | 动态星云粒子背景 |
| **CSS3 3D Transform** | 螺旋时间轴动画 |
| **Lottie** | 矢量动画效果 |
| **纯前端实现** | 零后端依赖，即开即用 |

## 🌟 核心亮点

1. **虚实难辨的视觉效果** —— 动态星云 + 粒子系统 + 扫描线 + 毛玻璃面板
2. **真实手势识别** —— 不是概念Demo，是真正能调摄像头的功能
3. **60年动漫数据库** —— 内置43部经典作品，从《铁臂阿童木》到《迷宫饭》
4. **螺旋时间树** —— 独特的星系螺旋布局，越靠近中心越古老
5. **零门槛使用** —— 一个HTML文件，双击即用
